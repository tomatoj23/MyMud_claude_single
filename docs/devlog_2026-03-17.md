#### 3. B4-GUI-003: 快捷键支持 ✅

**实现**:
- 实现 6 个全局快捷键：
  - `Ctrl+L` - 焦点到输入框
  - `Ctrl+K` - 清空输出区
  - `Ctrl+Q` - 退出应用
  - `Ctrl+S` - 保存游戏
  - `Ctrl+O` - 读取游戏
  - `F1/F2/F3` - 快速命令 (look/inventory/status)

**修改**:
- 添加 `_setup_shortcuts()` 方法
- 添加 `_quick_command()` 辅助方法
- 导入 `QKeySequence` 和 `QShortcut`

**测试**: 4 个单元测试，验证快捷键设置和快速命令功能

#### 4. B4-GUI-004: 存档/读档 GUI ✅

**实现**:

**SaveManager 类**:
- MessagePack 序列化游戏状态
- Gzip 压缩存档文件（压缩级别 6）
- SHA256 校验和验证
- 版本兼容性检查
- 存档元信息管理（JSON 格式）
- 支持列出、保存、读取、删除存档

**SaveDialog 对话框**:
- 列出现有存档（按时间倒序）
- 新建存档槽位（自动生成槽位号）
- 覆盖确认提示
- 存档名称输入

**LoadDialog 对话框**:
- 列出可用存档
- 显示存档详情（名称、时间、等级、位置、游戏时长、文件大小）
- 删除存档功能（带确认）
- 双击快速读取

**菜单栏集成**:
- 文件 → 保存游戏 (Ctrl+S)
- 文件 → 读取游戏 (Ctrl+O)
- 文件 → 退出 (Ctrl+Q)

**文件**:
- `src/engine/save/manager.py` (280 行)
- `src/engine/save/__init__.py`
- `src/gui/dialogs/save_dialog.py` (280 行)
- `src/gui/dialogs/__init__.py`
- `src/gui/main_window.py` (新增菜单和存档方法)

**依赖**: 使用 `msgpack` 库（已在 pyproject.toml 中声明）

**测试**: 5 个单元测试，验证存档管理器基础功能
