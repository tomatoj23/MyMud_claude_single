## 核心功能

### 1. 命令输入与输出

**输入方式**:
- 在输入框输入命令，按 `Enter` 或点击"发送"
- 支持命令历史记录

**输出特性**:
- 彩色文本显示（错误红色、对话蓝色、通知绿色）
- 富文本格式支持：
  - `**粗体**` - 显示为粗体
  - `*斜体*` - 显示为斜体
  - `` `代码` `` - 显示为代码块
  - `{red:红色文本}` - 显示为红色

**示例命令**:
```
look          # 查看周围环境
inventory     # 查看背包
status        # 查看状态
help          # 查看帮助
```

---

### 2. 快捷键

| 快捷键 | 功能 |
|:------|:-----|
| `Ctrl+L` | 焦点到输入框 |
| `Ctrl+K` | 清空输出区 |
| `Ctrl+S` | 保存游戏 |
| `Ctrl+O` | 读取游戏 |
| `Ctrl+,` | 打开设置 |
| `Ctrl+Q` | 退出游戏 |
| `F1` | 快速执行 look |
| `F2` | 快速执行 inventory |
| `F3` | 快速执行 status |

---

### 3. 存档系统

**保存游戏** (Ctrl+S):
1. 点击"文件" → "保存游戏"
2. 选择存档槽位或创建新存档
3. 输入存档名称
4. 点击"保存"

**读取游戏** (Ctrl+O):
1. 点击"文件" → "读取游戏"
2. 选择要读取的存档
3. 查看存档详情（时间、等级、位置）
4. 点击"打开"

**存档特性**:
- 支持多个存档槽位
- MessagePack + Gzip 压缩
- SHA256 校验和验证
- 显示游戏时长和文件大小

---

### 4. 主题系统

**切换主题**:
1. 点击"文件" → "设置" (Ctrl+,)
2. 在"外观"标签页选择主题
3. 点击"应用"或"确定"

**可用主题**:
- **暗色主题** (默认) - VS Code Dark 风格
- **亮色主题** - VS Code Light 风格

---

### 5. 设置界面

**打开设置** (Ctrl+,):
点击"文件" → "设置"

**外观设置**:
- 主题选择（暗色/亮色）
- 字体设置（字体、字号）
- 窗口大小（宽度、高度）

**编辑器设置**:
- 自动补全
- 命令历史（保存、数量）
- 输出行数限制
- 时间戳显示

**游戏设置**:
- 自动保存（启用、间隔）
- 音效开关
- 退出确认

---

### 6. 信息面板

**地图标签页**:
- 当前位置名称和描述
- 可用出口列表

**任务标签页**:
- 进行中的任务
- 可接的任务

**装备标签页**:
- 12 个装备槽位
- 当前装备显示

**背包标签页**:
- 物品列表（带数量）
- 负重信息

## 故障排除

### GUI 无法启动
```bash
# 检查依赖
pip install -e ".[dev]"

# 检查 Python 版本
python --version  # 需要 3.11+
```

### 存档无法读取
- 检查存档文件是否存在
- 验证存档版本兼容性
- 查看错误提示信息

### 主题未生效
- 重启应用
- 检查主题文件是否存在
- 尝试切换到另一个主题
