## ✨ 新功能演示

### 1. 界面显示
- 窗口正常显示，无黑屏问题
- 暗色主题自动应用
- 所有面板正确布局

### 2. 富文本格式
在输入框中尝试这些命令：

```
# 基础格式
这是 **粗体文本**
这是 *斜体文本*
这是 `代码文本`

# 颜色标记
{red:红色警告}
{green:绿色成功}
{blue:蓝色信息}
{yellow:黄色提示}

# 组合使用
**{red:重要警告}**: 你的 HP 已低于 {yellow:30%}！
```

### 3. 快捷键
试试这些快捷键：

| 快捷键 | 功能 |
|:------|:-----|
| `Ctrl+L` | 焦点到输入框 |
| `Ctrl+K` | 清空输出区 |
| `F1` | 快速执行 look |
| `F2` | 快速执行 inventory |
| `F3` | 快速执行 status |

### 4. 主题切换
1. 按 `Ctrl+,` 打开设置
2. 在"外观"标签页选择主题
3. 点击"应用"查看效果

### 5. 存档功能
- 按 `Ctrl+S` 保存游戏
- 按 `Ctrl+O` 读取游戏
- 支持多个存档槽位
