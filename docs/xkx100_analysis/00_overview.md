### 2.2 LPC语言特性

LPC（Lars Pensjö C）是MUD专用脚本语言：

| 特性 | 说明 | 示例 |
|:---|:---|:---|
| **继承** | 多重继承 | `inherit ROOM; inherit F_SAVE;` |
| **对象** | 一切皆对象 | `object me = this_player();` |
| **映射** | 关联数组 | `mapping exits = (["north": "/d/city/n"]);` |
| **函数指针** | 函数式编程 | `set("actions", (: call_other, ... :));` |
| **延迟调用** | 定时器 | `call_out("func", delay, arg);` |
| **影子** | 运行时修饰 | `shadow(who, new_obj);` |

### 2.3 核心机制分析

#### 2.3.1 对象创建流程

```lpc
// 1. 文件加载
/city/bingyindamen.c

// 2. 继承链加载
inherit ROOM;              → /inherit/room/room.c
  inherit CORE_OB;         → /inherit/core/object.c
    inherit F_DBASE;       → /inherit/feature/dbase.c
    inherit F_MOVE;        → /inherit/feature/move.c

// 3. create() 执行
void create() {
    set("short", "兵营大门");       // 设置属性
    set("long", "...");             // 描述
    set("exits", ([...]));          // 出口
    set("objects", ([...]));        // 初始对象
    setup();                         // 完成初始化
}
```

#### 2.3.2 命令处理流程

```
玩家输入 → process_input() → 命令解析 → 匹配命令对象
                                              ↓
                                        执行命令函数
                                              ↓
                                        返回结果/消息
```

#### 2.3.3 战斗系统流程

```
发起战斗 → combatd 注册 → heart_beat 触发
                                ↓
                    招式选择 → 命中判定 → 伤害计算
                                ↓
                          结果反馈 + 状态更新
```

### 3.2 设计理念差异

#### xkx100: "代码即内容"

```lpc
// 房间定义 = LPC代码文件
// 优点：灵活、可编程
// 缺点：内容与程序紧耦合

inherit ROOM;

void create() {
    set("short", "兵营大门");
    set("long", @LONG
这里是兵营大门...
LONG);
    set("exits", ([
        "south" : __DIR__"bingyin",
    ]));
    // 可编程逻辑
    set("no_clean_up", 1);
}

// 自定义逻辑
int valid_leave(object me, string dir) {
    if (dir == "south" && present("guan bing", this_object()))
        return notify_fail("官兵拦住了你的去路。\n");
    return ::valid_leave(me, dir);
}
```

#### 你的项目: "数据驱动"

```python
# 房间定义 = 数据文件(JSON/YAML)
# 优点：内容与程序分离
# 缺点：复杂逻辑需要额外机制

{
    "type": "room",
    "name": "兵营大门",
    "description": "这里是兵营大门...",
    "exits": [
        {"direction": "south", "target": "/city/bingyin"}
    ],
    "conditions": [
        {
            "type": "npc_present",
            "npc_id": "guan_bing",
            "block_direction": "south",
            "message": "官兵拦住了你的去路。"
        }
    ]
}
```

### 3.3 内容分离对比
